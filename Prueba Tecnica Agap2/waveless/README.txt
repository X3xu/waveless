
## Herramientas y Tecnologías

- **HTML5 & CSS3**: Estructura y estilos del proyecto.
- **Sass**: Preprocesador CSS que facilita la organización y mantenimiento del código.
- **JavaScript**: Funcionalidades interactivas del sitio.
- **Live Sass Compiler**: Para compilar automáticamente los archivos `.scss` a `.css`.  
  [Enlace a Live Sass Compiler](https://marketplace.visualstudio.com/items?itemName=ritwickdey.live-sass)
- **Live Server**: Para lanzar un servidor local y visualizar los cambios en tiempo real durante el desarrollo.  
  [Enlace a Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
